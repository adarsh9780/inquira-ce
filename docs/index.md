# Inquira CE

Inquira CE is a local-first AI data analysis app with three distribution channels:

- Python package (`pip install inquira-ce`)
- Desktop installers (macOS/Windows) via GitHub Releases
- Open-source docs and release notes on this site

## Quick Actions

- [Open latest release](https://github.com/adarsh9780/inquira-ce/releases/latest)
- [Browse all releases](https://github.com/adarsh9780/inquira-ce/releases)
- [Install with pip](https://pypi.org/project/inquira-ce/)

## What this site includes

- Download links for wheel and desktop assets
- Setup and development guides
- Release and CI workflow documentation
- Project roadmap and contribution guide

If this is your first time here, start with [Downloads](./downloads.md) and then move to [Install](./install.md).

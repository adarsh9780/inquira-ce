# Inquira Community Edition (CE)

Inquira CE is a local-first analytics engine built for solo data analysts, students, and developers who need ad-hoc analysis without cloud lock-in or artificial limits.

## What's Included

- **Cost:** 100% Free ($0).
- **Access:** Zero friction. No login or credit card required to start analyzing.
- **Workspaces:** **Unlimited Local Workspaces** per machine, with unlimited datasets per workspace.
- **LLM Support:** Bring Your Own Key (BYOK) for OpenAI, Anthropic (testing phase), OpenRouter (most tested), or Ollama (partially tested).
- **Support:** Community-supported bug tracking and discussions.
